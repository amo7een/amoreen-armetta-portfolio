## 👋 Welcome!

I built this portfolio using plain HTML, CSS, and vanilla JS with a no-install, zero-build philosophy. This repository serves as a practical showcase of my approach to **docs-as-code**:
* **User-Centric Design:** The deployment guide below is written to be highly accessible, proving that even technical workflows can be made intuitive for anyone.
* **Low-Maintenance Systems:** By avoiding heavy frameworks, this site loads instantly, requires zero dependency updates, and is incredibly easy to scale or edit.
* **Information Architecture:** The repository structure is kept intentionally lean, clean, and self-documenting.
